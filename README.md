# 古月浮屿｜Guyue Floating World

> 一张书桌连接星河。

古月浮屿不是传统博客，而是一个正在生长的个人数字世界。

它以内容为星体，以区域为空间，以关系为星线，以时间为河流，以权限为边界，以规则为自然法则，以 AI 为灯塔。

## V1 技术栈

```text
Next.js
React
TypeScript
Tailwind CSS
Framer Motion
MDX / Markdown
JSON 数据协议
Zod
Fuse.js
Shiki
Mermaid
静态部署
```

## V1 原则

```text
静态优先
协议先行
无 AI 可运行
AI 可插拔
公开与私密构建隔离
Markdown / JSON 可导出
```

## 本地开发

```bash
npm install
npm run validate:world
npm run check:public
npm run dev
```

## 核心目录

```text
src/app/          页面路由
src/components/   世界组件
src/lib/          数据协议、读取、校验、搜索
data/             世界数据
content/          Markdown / MDX 内容
scripts/          构建前守门脚本
```

## 重要边界

V1 不做 3D 元宇宙，不做 Avatar，不做多人实时空间，不做完整 RAG，不做 CMS 后台。
宇宙感首先来自空间结构、时间流动、节点关系、法则边界、入口仪式和长期存在。
