# AI 边界进化方向

AI 是灯塔不是太阳。但随着世界成熟，灯塔可以从低光进化为全光。

## 进化路径

1. **低光模式（当前）**：静态索引 + 预设问答，不调用真实 AI
2. **半光模式**：搜索 + 推荐引擎，仍不生成内容
3. **全光模式**：接入真实 AI provider，基于公开上下文生成回答

## 边界契约演进

无论进化到哪个阶段，核心契约不变：AI 只读公开上下文，不写入世界，不读取私密内容。

## 成本控制

全光模式需要成本控制：每日请求上限、缓存策略、降级阈值。超限后自动回退到低光模式。

## 当前状态

古月浮屿当前处于低光模式。搜索 API 和推荐引擎已就绪，问答 API 已创建但未接入真实 provider。


## 边界测试

无论进化到哪个阶段，都需要自动化边界测试。测试用例包括：AI 是否只读取了 public 数据、AI 响应中是否泄漏了私密内容、降级机制是否在超限时生效。这些测试纳入 check:daily 门禁，确保边界契约不被意外破坏。
