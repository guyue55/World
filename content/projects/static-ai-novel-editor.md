# 静态 AI 小说编辑器

一个纯静态前端的小说创作工具设想。

目标：

- 浏览器本地保存配置
- 用户自行配置 API Key
- 支持小说编辑器
- 支持深色模式
- 支持多种风格按钮
- 移动端可用

## 为什么是静态前端

小说创作的核心是文字和结构，不需要服务端数据库。配置和草稿保存在浏览器本地存储中，API Key 由用户自行填入，不经过任何中间服务器。这样做的好处是部署成本极低、隐私可控、离线时仍可编辑已有内容。
