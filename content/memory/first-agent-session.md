# 首次 AI Agent 协作

第一次让 Codex 独立完成一次工程动作是一个安静的下午。目标很小：给 nodes.json 添加一个字段并跑通门禁。整个过程二十分钟，其中十五分钟是等待，五分钟是审阅。

## 与以往的不同

以往 AI 更多像顾问：给出建议、写出代码片段，最终由人贴回工程。这一次它自己拉分支、跑 test、写 commit。虽然还需要审阅，但工作面已经交给它了。

## 那种平静

没有预想中的兴奋，反而是一种平静：世界多了一个能干活的居民，日常从此不必一人扛。这种平静后来成为 codex-agent-workflow 的第一条纪律：不把 AI 当作事件，把它当作日常。

## 与提示词卫生的关系

第一次协作也暴露了边界问题：如果没有提示词卫生，可能会给它读到 owner-only 的数据。这次会话催生了 ai-prompt-hygiene 这条规则。

## 那次会话之后

从那次开始，Codex 逐渐承担了世界维护里最枯燥的部分。人负责判断和取舍，AI 负责执行和记录。分工稳定之后，创作变得可持续。
