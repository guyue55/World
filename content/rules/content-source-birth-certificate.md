# 内容出生证明

每个节点都应该知道自己从哪里来。

建议记录：

- source：manual / chat / upload / import / github / ai-assisted。
- originalInput：原始输入或来源摘要。
- generatedBy：creator / rule / ai。
- reviewed：是否人工确认。
- publicVersionOf / privateSourceOf：公开版和私密源的关系。

这能避免多年后忘记某个节点是手写、导入、AI 辅助，还是从对话中提取出来的。

## 出生证明的价值

当世界积累了上百个节点后，来源追溯变得至关重要。一个 AI 辅助生成的节点如果被当作手写原文引用，会造成信任错位。出生证明不是学术引用规范，而是最轻量的来源标记：每个节点只需记录 source 和 reviewed 两个字段，就能在未来快速筛选出"哪些需要复查"。
