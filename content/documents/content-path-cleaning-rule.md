# 正文路径清理规则

## 维护价值
这个节点把一个容易散落在对话里的判断固定下来，方便以后从地图或档案馆重新找到。记录读取 Markdown 正文时的路径安全边界 它位于档案馆，主要任务是把维护材料整理成以后能查到的说明。读者来到这里时，不需要理解所有历史阶段，只要能抓住这一页提供的一个判断：路径清理。

在公开世界里，这个节点要服务三个对象：第一次访问的人、未来维护的人、以及只读低光模式下的灯塔。对第一类人，它提供一句能带走的解释；对第二类人，它保留可检查的依据；对灯塔，它提供可以引用但不能越权扩写的公开上下文。

## 低门槛说明
它被放入 Phase 19 的路径，是因为世界已经从 130 个节点进入更大规模。节点数量增加后，单个页面必须更清楚地说明自己为什么存在。这里的复核重点是：字段是否完整、路径是否稳定、来源是否可回溯。如果这些问题回答不出来，这页就只是在堆叠内容。

它和同区域节点的关系不是机械连线，而是让读者能从抽屉、索引、封签之间移动。相关标签包括：phase19、archive、contentPath、security、read。这些标签会进入档案馆筛选和灯塔搜索，但最终仍以正文解释为准。

## 回访提示
维护者复看时，应该能从标题和摘要直接判断它是否仍然放在正确区域。 本节点保持 public 可见，只使用公开语境，不包含 vault、sealed、silent 或家庭私密内容。若未来接入真实 AI Provider，它也只能作为公开上下文候选，输出必须进入审核队列，不能自动修改世界源文件。
