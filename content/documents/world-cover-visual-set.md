# 世界视觉套图与封面系统

古月浮屿的视觉不应每页各自为政。

视觉资产需要遵守一组锚点：

- 纸上宇宙。
- 书桌星河。
- 自然轻科技。
- 低光灯塔。
- 温柔档案馆。

封面、图标、页面背景和项目图应尽量从这些锚点延展，而不是随流行风格摇摆。

## 锚点如何使用

每个区域可以有自己的主色调倾向，但必须从这五个锚点中选取基调。例如，技术区域偏向"自然轻科技"，记忆区域偏向"温柔档案馆"，工坊区域偏向"书桌星河"。新增封面时先确认它属于哪个锚点，再开始设计。

## 为什么不用流行风格

流行风格的生命周期很短。如果世界视觉跟随每年的设计趋势，几年后就会显得过时。锚点不是风格，而是世界本身的气质——它们应该像世界的语言一样稳定，只在细节上演变。

## 补充说明

这个节点是古月浮屿世界的一部分，通过节点护照、关系星线和推荐路径与世界其他内容相连。访客可以通过地图、路径、档案馆或灯塔推荐到达这里。节点内容遵循世界协议：公开层只展示可公开的信息，所有变更通过门禁验证。世界在持续生长，这个节点也会随时间补充更多实践经验和边界思考。
