# Homelab 网络搭建笔记

一张书桌连接星河，而在星河之下，是一根根网线和一台台设备。

## 网络拓扑

家庭网络的核心是路由器。从光猫出来，经过路由器分配到各个设备。如果设备多，还需要交换机扩展端口。

```text
光猫 -> 路由器(WAN) -> 路由器(LAN) -> 交换机 -> 各设备
```

## 关键配置

1. 光猫改桥接模式，让路由器拨号
2. 路由器开启 DHCP，分配 IP 段
3. 重要设备绑定静态 IP
4. 端口转发按需配置，不暴露不必要的服务

## 排障方法

- `ping` 测试连通性
- `traceroute` 追踪路由路径
- `nslookup` 检查 DNS 解析
- `netstat -tulpn` 查看端口占用

## 无线网络

2.4G 穿墙好但速度慢，5G 速度快但穿墙差。Mesh 组网是解决大户型覆盖的方案。


## 安全建议

家庭网络的安全往往被忽视。改掉路由器默认密码是第一步，关闭 WPS 是第二步。如果路由器支持访客网络，把 IoT 设备隔离到访客网络中，可以有效降低被入侵后的影响范围。定期检查固件更新，保持路由器软件最新。
