# 世界协议入门指南

世界协议是古月浮屿的完整规范，定义了数据结构、权限模型、AI 上下文契约和导出格式。

## 五层数据模型

世界由五层数据组成：节点（Node）、区域（Area）、关系（Relation）、路径（Path）和事件（Event）。每一层都有明确的字段定义和验证规则，确保数据一致性。

## 权限分层

世界采用三层权限：公开层（public）、私密层（private）和封存层（sealed）。权限由后端 middleware 和 API 路由守门，前端只控制体现，不作为安全边界。

## AI 边界

AI 是灯塔不是太阳。AI 只读公开上下文，不写入世界，可降级。灯塔搜索和问答 API 使用 GET + force-static，避免写入信号。

## 导出格式

世界可一键导出为 Markdown + JSON + 媒体文件，不依赖任何平台。导出只包含公开层数据，私密层永不进入导出包。

## 机器可读索引

`public/world-index.json` 是世界的机器可读索引，包含所有公开节点的元数据。AI 工具和外部系统可通过该索引理解世界结构。
