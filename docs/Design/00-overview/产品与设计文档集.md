# 古月浮屿｜产品与设计文档集

> 本文档集继续承接前面的《世界宪章》《世界手册》《世界规则》《AI 守则》《V1 实施蓝图》《术语词典》。

> 原则：详细、可执行、宁冗余不错漏。

## 文档清单

1. [《古月浮屿｜产品总览》](./01_古月浮屿_产品总览.md)  
   定义产品定位、目标用户、价值主张、核心闭环、差异化和成功指标。

2. [《古月浮屿｜PRD 产品需求文档》](./02_古月浮屿_PRD产品需求文档.md)  
   用产品需求文档方式描述 V1/V2/V3 的目标、用户故事、功能需求、优先级、边界和验收标准。

3. [《古月浮屿｜信息架构与导航设计》](./03_古月浮屿_信息架构与导航设计.md)  
   定义站点结构、主区域、路由、路径、地图、档案视图、反迷路系统。

4. [《古月浮屿｜交互体验设计文档》](./04_古月浮屿_交互体验设计文档.md)  
   定义进入世界、地图、节点、时间河、搜索、AI 灯塔、创世台、仪式感、空状态、错误状态。

5. [《古月浮屿｜视觉设计系统》](./05_古月浮屿_视觉设计系统.md)  
   定义美学方向、颜色、字体、布局、动效、图标、卡片、移动端视觉、禁区。

6. [《古月浮屿｜内容系统与编辑规范》](./06_古月浮屿_内容系统与编辑规范.md)  
   定义内容类型、节点护照、命名、写作模板、精选机制、公开版本、生命周期和内容质量标准。

7. [《古月浮屿｜技术架构设计》](./07_古月浮屿_技术架构设计.md)  
   定义前端架构、目录结构、数据模型、构建管线、搜索、权限隔离、AI 插拔和部署策略。

8. [《古月浮屿｜路线图与版本计划》](./08_古月浮屿_路线图与版本计划.md)  
   定义 V0/V1/V2/V3/V4/V5 的阶段目标、功能边界、验收指标和风险控制。
