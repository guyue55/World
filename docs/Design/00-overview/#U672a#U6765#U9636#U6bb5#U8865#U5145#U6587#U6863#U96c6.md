# 古月浮屿｜未来阶段补充文档集

> 说明：本组文档补齐 V4 / V5 / V6 阶段需要的关键设计文档。
> 原则：先形成基础版本，后续在开发过程中持续修订完善。
> 注意：这些文档服务于未来阶段，不要求 V1 全部实现。

## 文档清单

1. [16_古月浮屿_私密档案系统设计.md](./16_古月浮屿_私密档案系统设计.md)
   设计 private / family / partner / vault / sealed / silent 等私密层结构，明确公开层与私密层的物理隔离、权限、导出与 AI 边界。

2. [17_古月浮屿_家庭与孩子内容规范.md](./17_古月浮屿_家庭与孩子内容规范.md)
   重点覆盖家庭灯火、孩子年轮、爱的小屋、共同记忆、未来孩子权利和默认隐私策略。

3. [18_古月浮屿_时间胶囊产品设计.md](./18_古月浮屿_时间胶囊产品设计.md)
   设计写给未来自己、孩子、家人或项目未来状态的时间胶囊系统。

4. [19_古月浮屿_年度世界册设计.md](./19_古月浮屿_年度世界册设计.md)
   设计年度回顾、年度册结构、年度候选节点、导出 PDF / Markdown / 静态页面等机制。

5. [20_古月浮屿_主题展览系统设计.md](./20_古月浮屿_主题展览系统设计.md)
   设计 AI Agent 探索展、旅行远方展、失败项目遗迹展、世界创世展等专题展览系统。

6. [21_古月浮屿_导出与传承系统设计.md](./21_古月浮屿_导出与传承系统设计.md)
   设计公开世界包、个人完整备份包、家庭纪念包、孩子成长包、旅行纪念包、项目作品包、长期传承说明。

7. [22_古月浮屿_高级世界体验设计.md](./22_古月浮屿_高级世界体验设计.md)
   作为 V6 候选，设计 2.5D / Canvas 星图 / 轻量 3D / 声音层 / 多尺度缩放等高级体验，但明确不进入 V1 核心范围。

```text
  → 2.5D / 3D / 声音 / 多尺度体验
```
