# Round 02｜深度体验兑现 / 数字宇宙体验成型

## Mission

Round 02 moves word-life from an accessible world-entry website into a stronger personal digital universe experience.

```text
First round solved: repository, structure, access, documentation
Second round solves: experience depth, visual impact, interaction, data-driven views, real theme system
```

## Principles

```text
1. Work only inside word-life/
2. Every batch must include data + docs + check
3. Pages assemble; features model; components render
4. No private archive raw content exposure
5. AI is a lighthouse, not the sun
6. No productionLive claims
7. WebGL/3D must not be a hard dependency in Round 02
8. check:round2 and check:local must pass before closure
```

## Non-goals

```text
- No real backend integration
- No real login/auth system
- No private archive raw reads
- No automatic publish/delete/visibility changes
- No production deployment promise
- No heavy 3D engine as first dependency
```

## Stages

```text
Stage 01: Visual Foundation
Stage 02: World Map
Stage 03: Content Constellation
Stage 04: Time River + Memory Graph
Stage 05: AI Lighthouse Workbench
Stage 06: Theme System + Final Closure
```

## Completion Standard

```text
1. Home first screen no longer feels like a normal blog
2. World map becomes the navigation hub
3. Content can be explored by node / constellation / path
4. Time river expresses V1-V6 and experience evolution
5. Memory graph keeps public/private redaction boundaries
6. AI lighthouse has suggestion flow, risk level, audit queue
7. Theme switching changes layout and components, not only color
8. Mobile has no horizontal overflow and no nav blocking
9. check:round2 and check:local pass
10. docs and data fully record the work
```
