# V5｜真实内容版

## Mission

V4 已经让项目“像世界”，V5 的任务是让世界真正长出内容：真实文章、真实照片、项目记录、生活记录、时间事件、主题展览和阅读路径。

## Scope

```text
Stage 01: 真实内容种子与内容域
Stage 02: 照片、项目与生活记录内容化
Stage 03: 时间河、主题展览与阅读路径真实化
Stage 04: 内容审查、封版与 V6 交接
```

## Rules

```text
- 不使用 private raw content。
- placeholder 不伪装成真实内容。
- 真实内容可以是精选样例，但必须具有真实栏目、真实用途和可运营结构。
- productionLive remains false.
```
