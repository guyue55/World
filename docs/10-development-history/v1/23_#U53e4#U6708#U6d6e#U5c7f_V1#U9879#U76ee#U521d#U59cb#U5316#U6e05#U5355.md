# 古月浮屿｜V1 项目初始化清单

> 目标：从文档阶段进入 V1 开发阶段，建立第一座可访问、可探索、可维护的核心城。
> 原则：先数据，后页面；先静态，后交互；先可用，后动效；先公开世界，后私密深层；先低光灯塔，后完整 AI。

---

## 0. 初始化前确认

### 0.1 守则确认

开发前必须再次确认：

- [ ] 古月浮屿不是普通博客，而是个人数字世界
- [ ] AI 是灯塔，不是太阳
- [ ] 公开层不是完整世界
- [ ] 私密内容默认受保护
- [ ] 无 AI 时世界仍能运行
- [ ] V1 不是完整宇宙，只是第一座核心城
- [ ] 前台可以浪漫，后台必须清醒
- [ ] 数据可导出，世界可迁移

### 0.2 V1 范围确认

V1 做：

- [ ] 首页 / 世界入口
- [ ] 世界地图 Atlas
- [ ] 节点系统 Node
- [ ] 节点详情页
- [ ] 时间河 Timeline
- [ ] 档案馆 Archive
- [ ] 精选路径 Paths
- [ ] AI 灯塔低光模式 Ask
- [ ] 关于古月 About
- [ ] 世界宪章 Manifesto
- [ ] 基础本地搜索
- [ ] 第一批真实内容
- [ ] README
- [ ] public/private 基础隔离

V1 不做：

- [ ] 完整 3D 宇宙
- [ ] 完整 AI RAG
- [ ] 完整后台
- [ ] 多用户权限
- [ ] 完整爱的小屋
- [ ] 完整孩子年轮
- [ ] 完整历史剧场
- [ ] 声音系统
- [ ] 社交系统

## 1. 仓库初始化

### 1.1 创建项目

推荐命令：

```bash
npx create-next-app@latest guyue-floating-world \
  --ts \
  --tailwind \
  --eslint \
  --app \
  --src-dir \
  --import-alias "@/*"
```

进入项目：

```bash
cd guyue-floating-world
```

### 1.2 推荐安装依赖

```bash
npm install framer-motion clsx tailwind-merge lucide-react
npm install fuse.js
```

可选：

```bash
npm install gray-matter reading-time
npm install zod
```
