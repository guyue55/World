# 真实世界-R4｜创世台 / 工作台真实化规划

R4 的目标不是继续扩张世界观，而是让 R3 已经进入世界的真实节点可以被主人长期维护。

## 阶段规划

1. 创世台信息架构与收集箱：建立星尘港、草案区、节点操作语义和主人入口。
2. 节点管理、权限检查与内容债务：建立节点管理台、权限守门、内容债务和孤岛节点修复。
3. 维护闭环、审计日志与导出备份：建立日常轻仪式、审计日志、导出计划和世界健康度。
4. 质量门禁、隐私边界与 R5 交接：完成 R4 聚合检查、隐私边界扫描、文档和 R5 AI 灯塔交接。

## 扩展任务

星尘港收集箱、节点管理台、权限风险守门、内容债务队列、孤岛节点修复、世界健康度、每日轻仪式、低心力维护、审计日志、危险操作确认、导出备份计划、R5 AI 灯塔交接。

## 设计原则

前台浪漫，后台清醒，档案可靠。外面是浮屿，里面是创世台。
