# 古月浮屿｜V1 世界演化策略

> 阶段：V1
> 性质：世界长期生长策略
> 目标：让世界可以持续生长，但每次生长都有理由、事件和可回退路径。

## 1. 演化策略文件

```text
data/evolution-policy.json
```

## 2. 变化类型

```text
minor-content
node-publication
relation-core-change
visibility-policy-change
schema-breaking-change
private-structure-change
ai-boundary-change
future-extension-activation
```

## 3. 重大变化必须具备

```text
snapshot plan
WorldEvent
review
check:world-core
```

## 4. 最终原则

```text
世界可以生长，
但不能无记录地变形。
```
