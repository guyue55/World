# 古月浮屿｜V1 视觉与交互缺陷登记册

> 阶段：V1
> 性质：真实 QA 缺陷台账
> 目标：把视觉和交互问题从“感觉不对”变成可跟踪、可关闭的缺陷项。

## 1. 文件

```text
data/visual-interaction-defect-register.json
```

## 2. 初始 watch 项

```text
VIS-001 桌面端内容可能过窄
INT-001 标签切换内容变化但选中态可能不同步
INT-002 折叠/展开类模块可能不可点击或缺少反馈
CNT-001 首页缺少有价值配图或真实内容代表性不足
```

## 3. 状态规则

```text
watch  等待真实浏览器复现
open   已确认存在，需要修复
closed 已修复并有验证说明
```

## 4. 核心判断

```text
真实 QA 发现的问题必须进入缺陷登记，
而不是靠记忆处理。
```
