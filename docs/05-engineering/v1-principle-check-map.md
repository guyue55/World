# 古月浮屿｜V1 原则到检查映射

> 阶段：V1
> 性质：原则可执行化
> 目标：让核心原则能落到检查链路，而不是停留在口号。

## 1. 映射文件

```text
data/principle-check-map.json
```

## 2. 映射内容

每条核心原则至少包含：

```text
principle
checks
evidence
```

## 3. 示例

```text
AI 是灯塔，不是太阳。
→ check:ai-boundary
→ data/ai-boundary-policy.json
```

## 4. 最终原则

```text
原则必须能落到检查链路，
否则只是口号。
```
