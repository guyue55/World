# 古月浮屿｜V1 世界内核

> 阶段：V1
> 性质：世界骨架总聚合
> 目标：把协议、规则、边界、索引、扩展、降级和导出统一收束为“世界内核”。

## 1. 核心原则

```text
内核不负责华丽，
内核负责世界能长期成立。
```

## 2. 内核文件

```text
data/world-kernel-manifest.json
src/lib/world-kernel.ts
```

## 3. 内核负责检查

```text
世界骨架健康度
世界不变量
AI 边界
投影契约
扩展注册表
空间协议
语义层
关系规则
降级策略
导出契约
未来兼容矩阵
```

## 4. 执行命令

```bash
npm run check:kernel
npm run kernel:report
```

## 5. 最终原则

```text
页面会变化，
体验会变化，
技术会变化，
但内核必须长期稳定。
```
