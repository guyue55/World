# 古月浮屿｜V1 世界地基准入门

> 阶段：V1
> 性质：世界骨架阶段门禁
> 目标：确认世界/宇宙骨架已经足够稳定，可以进入后续产品化和内容种子阶段。

## 1. 核心原则

```text
冻结的是骨架底线，
不是未来想象力。
```

## 2. 准入数据

```text
data/foundation-freeze.json
data/change-admission-policy.json
data/development-gates.json
data/foundation-quality-radar.json
```

## 3. 执行命令

```bash
npm run check:foundation
npm run foundation:print
```

## 4. 地基通过条件

```text
world kernel blocking errors = 0
foundation quality score >= 85
frozen foundation items >= 8
```

## 5. 最终原则

```text
先让地基稳定，
再让世界变得复杂。
```
