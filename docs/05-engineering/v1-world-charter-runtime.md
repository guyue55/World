# 古月浮屿｜V1 世界宪章运行化

> 阶段：V1
> 性质：原则到运行时的映射协议
> 目标：让世界宪章不只停留在文档，而是进入检查、治理和未来扩展护栏。

## 1. 运行化文件

```text
data/world-charter-runtime.json
```

## 2. 核心原则

```text
世界宪章不能只写在文档里，
它必须进入检查、治理和未来扩展护栏。
```

## 3. 宪章运行映射

```text
古月浮屿不是传统博客 → world-manifest / world-kernel
AI 是灯塔，不是太阳 → ai-boundary-policy / ADR-0003
公开层不是完整世界 → permissions / export-contract / ADR-0004
无 AI 时世界仍能运行 → fallback-strategy / low-light mode
世界可以生长 → evolution-policy / world-events / snapshot-policy
未来扩展通过注册点进入世界 → extension-registry / maturity model
```

## 4. 最终原则

```text
原则必须能落到运行时，
否则只是口号。
```
