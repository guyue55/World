# 古月浮屿｜V1 能力插槽与未来阶段兼容

> 阶段：V1
> 性质：未来阶段兼容设计
> 目标：让 V2 到 V6 有清晰接口，而不是未来重构 V1 骨架。

## 1. 能力插槽文件

```text
data/capability-slots.json
```

## 2. 能力插槽不是承诺

能力插槽表示：

```text
未来可以从这里长出来。
```

但不表示：

```text
V1 必须实现它。
```
