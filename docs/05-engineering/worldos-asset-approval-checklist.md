# WorldOS 资产审批清单

## 适用范围

图片、图标、字体、音频、音乐、动效素材、截图和外部数据资产进入项目之前使用本清单。

## 必填信息

- 来源：项目自制、公开授权、系统生成、第三方库或人工拍摄。
- 授权：许可证、使用范围、署名要求、是否可商用。
- 用途：服务哪个场景、节点、氛围或文档。
- 体积：原始大小、压缩后大小、是否进入首屏。
- 降级：加载失败、reduced-motion、静音模式、低带宽下如何表现。
- 存放位置：不要散落到未知目录。

## 当前策略

- 静态图片优先放在资产清单覆盖的目录中。
- 音频默认不使用外部文件；当前声景使用 Web Audio 合成短音，默认静音。
- 字体和大媒体不进入首屏，除非有明确收益和预算。
- 来历不明资产不得进入公开页面。

## 检查命令

```bash
npm run check:performance-budget
npm run check:dependency-hardening
npm run check:scene-qa
```

## 禁止项

- 无来源、无授权、无用途的资产进入项目。
- 默认自动播放声音。
- 为单个效果引入大型依赖或重型依赖。
- 在公开页面加载私密素材、未发布草稿或 owner-only 资源。

## 验收

资产必须让世界更清晰、更有氛围或更易理解。若只是装饰、拖慢加载或增加维护负担，应拒绝。
