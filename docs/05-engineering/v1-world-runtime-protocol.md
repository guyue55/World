# 古月浮屿｜V1 世界运行态协议

> 阶段：V1
> 性质：世界长期运行协议
> 目标：让世界不只是能构建，也能被启动、观测、降级、恢复和维护。

## 1. 协议文件

```text
data/world-runtime-protocol.json
```

## 2. 运行态

```text
drafting
checking
ready
published
degraded
blocked
archived
```

## 3. 核心原则

```text
世界不是一次构建物，
而是需要启动、观测、恢复和维护的长期生命体。
```

## 4. 关键规则

```text
blocked 状态不得发布。
degraded 状态必须保留核心导航。
published 状态不得包含私密内容。
重大变更前必须进入 archived/snapshot 流程。
ready 不等于 finished，只表示骨架健康。
```
