# ADR-0003｜AI 是灯塔而不是太阳

## 状态

Accepted

## 背景

AI 能帮助总结、导航、生成草案、建议关系，但不能替代创作者做最终判断，也不能绕过权限和发布流程。

## 决策

AI 在古月浮屿中定位为：

```text
灯塔
草案助手
导航助手
关系建议者
```

AI 不允许：

```text
自动发布
自动改 visibility
读取 vault / sealed / silent
覆盖原文
将想象层当作事实层
```

## 后果

好处：

```text
AI 可用但不失控
私密边界清晰
无 AI 时世界仍可运行
```

代价：

```text
AI 相关内容需要审核
自动化程度刻意受限
```

## 结论

```text
AI 可以照亮路径，
但不能决定方向。
```
