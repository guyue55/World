# 九张视觉套图素材映射

## 总目标

从“游戏化 AI 概念图”转向“真实、震撼、近距离接触的个人数据宇宙”。每张图都必须同时具备：

- 近景真实材质：石头、木纹、纸张、植物、水、黄铜、玻璃、泥土、苔藓。
- 宏大背景尺度：银河、星云、巨大行星、云海、山体、天文台、河谷。
- 克制数据秩序：星图线、光谱线、天文坐标、微弱数据尘埃。

## 01 世界入口

素材组合：石拱门 / 山巅云海 / 银河 / 巨大行星。

设计重点：低机位、石阶贴近镜头，门后宇宙不能像传送门特效，要像真实天象突然压入现实。

## 02 世界地图

素材组合：羊皮纸星图 / 黄铜星盘 / 望远镜 / 雪山银河。

设计重点：地图不是游戏 UI，而是天文制图桌。数据世界的“规律感”来自坐标、刻度和星轨。

## 03 典藏星馆

素材组合：古典图书馆 / 高窗星空 / 古书近景 / 微尘。

设计重点：知识归档感，真实木材和纸张优先，弱化魔法光。

## 04 创作工坊

素材组合：木桌 / 键盘 / 手稿 / 茶杯 / 植物 / 窗外银河。

设计重点：人真的坐在这里创作，窗外宇宙是“精神尺度”而不是装饰背景。

## 05 生活浮屿

素材组合：悬崖巨岩 / 苔藓草根 / 瀑布水雾 / 木屋花园 / 云海湖泊。

设计重点：悬空岛屿要像真实地质体，必须有重量、岩层、水、植物根系。

## 06 家族花园

素材组合：玻璃温室 / 花草水珠 / 暖灯 / 星月 / 家人剪影。

设计重点：温柔真实，不做童话花园。亲情是灯光和空间关系，不是符号堆叠。

## 07 时间长河

素材组合：河面近景 / 湿石 / 银河倒影 / 旧照片 / 怀表 / 古桥钟楼。

设计重点：时间要从“水的真实反射”和“物件的旧质感”表达。

## 08 AI 灯塔

素材组合：天文台 / 望远镜 / 灯塔 / 海崖云海 / 星轨 / 光谱线。

设计重点：AI 是观测与指引，不是机器人或赛博中控台。

## 09 生长四季

素材组合：树干年轮 / 春花 / 夏绿 / 秋叶 / 冬雪 / 山谷星空。

设计重点：四季不是彩色拼贴，而是同一个真实空间的时间变化。
