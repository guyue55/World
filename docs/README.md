# 文档中心

> [!IMPORTANT]
> 当前开发唯一入口是 [WorldOS Reality-First 一次性 Goal 提示词](00-overview/worldos-reality-first-one-shot-goal-prompt-2026-07-10.md)。其总控、验收、架构、执行计划和账本共同取代旧 M8-M30 的完成口径；旧 `ultimate`、Phase、M 与 RC 文档只作历史参考。

本文档中心按长期 GitHub 项目治理方式组织，避免把“代码开发前文档”和“开发过程文档”混在一起。

## 目录

```text
00-inception/              最初沟通、原始需求、原始素材
01-world-design/           世界观、世界规则、AI 守则、术语
02-product-design/         PRD、信息架构、交互、视觉、内容规范
03-engineering-architecture/ 技术架构、系统设计、数据模型、接口、安全、QA
10-development-history/    第一轮 V1/V2/V3/V4/V5/V6 与体验兑现开发过程
20-research/               开源、数字宇宙、采纳矩阵等调研
30-assets/                 视觉素材与图像资产
90-archive/                历史报告、归档说明、旧包索引
```

## 规则

- 开发前形成的文档放在 `00/01/02/03`。
- 开发过程中每一轮、每一版、每一批说明放在 `10-development-history/`。
- 研究资料放在 `20-research/`。
- 图片素材放在 `30-assets/`。
- 旧报告和归档记录放在 `90-archive/`。
- 后续新增文档必须先判断阶段属性，不得混放。

**重点：**


### V4｜世界化重构版

**目标：** 把“普通博客形态”真正重构为“个人数字世界形态”。

这是当前最应该进入的版本。

**重点：**

```text
首页世界入口化
Atlas 世界地图主入口
导航从菜单改为地图 / 星图 / 浮岛路径
内容从文章卡片改为世界节点
时间线从列表改为时间河
分类从栏目改为主题展览 / 星域 / 花园 / 浮岛
AI 灯塔从功能页改为世界中的导航者 / 观测站
整体视觉从网页布局改为宇宙 / 浮岛 / 花园 / 档案馆式空间
移动端保持可用但不牺牲世界感
```

**终点：**

```text
第一眼不再像普通博客，而像一个可以进入、探索、漫游、阅读和沉淀记忆的个人数字世界。
```
