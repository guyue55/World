# 外部真实素材源与使用策略

本素材库用于“古月浮屿｜数据世界/宇宙视觉套图”重构，优先采集真实宇宙、星空、星辰、行星、银河、自然风景、季节风景、花草树木、空中花园/悬空岛屿参考等素材。

## 素材源优先级

1. NASA Image and Video Library
   - 优先级：S
   - 用途：宇宙、星云、行星、银河、深空、航天器、地球影像、视频。
   - 备注：通常适合教育/信息用途；使用时注明 NASA 来源，不暗示 NASA 背书；注意第三方版权标注、NASA logo、可识别人像限制。

2. ESO Public Images / Videos
   - 优先级：S
   - 用途：银河、星云、天文台、星空、深空视频。
   - 备注：多数公开素材为 CC BY 4.0，必须保留完整 credit。

3. Wikimedia Commons
   - 优先级：A
   - 用途：真实自然风景、山海云雾、庭院花园、植物、季节、部分天文图。
   - 备注：需逐项读取 license、artist、attributionRequired、creditLine。

4. Pexels / Pixabay
   - 优先级：B
   - 用途：自然、森林、花草、云海、河流、季节、视频素材。
   - 备注：免费可用但有平台自定义限制；不要把未修改素材当作独立素材包二次分发，不用于商标/品牌标识。

5. Unsplash
   - 优先级：B-
   - 用途：自然真实摄影参考。
   - 备注：适合作为视觉参考和网站背景，但仍建议保留摄影师来源，避免直接作为可转售素材。

## 采集原则

- 只采集真实摄影、真实天文图像、真实自然素材；不采集明显 AI 图、游戏概念图、卡通图。
- 每条素材必须保留：source、title、author/credit、license、origin_url、download_url、用途场景、风险标记。
- 能直接落盘的素材，放入 `assets/raw/<source>/<category>/`；当前包先提供可复现清单和下载脚本。
- 外部二进制素材下载失败时，保留查询 URL、API 参数、失败日志，便于本地联网环境继续执行。
