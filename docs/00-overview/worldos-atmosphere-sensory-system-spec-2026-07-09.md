# WorldOS 氛围与感官系统规格

> [!IMPORTANT]
> 氛围系统不是背景装饰，而是世界运行状态的感官投影。它统一管理光线、时间、季节、粒子、声音状态和 AI 状态。

## 1. 目标

让 WorldOS 的每个场景拥有可感知但不喧宾夺主的环境层，使访问者感到世界正在运行，而不是静态页面被套上动效。

## 2. 输入状态

| 状态 | 来源 | 用途 |
| --- | --- | --- |
| `sceneId` | 路由 / 场景注册表 | 决定场景基础氛围 |
| `dayPeriod` | 本地时间或手动默认 | 影响光线和语气 |
| `season` | 日期或内容状态 | 影响色温和材质 |
| `aiStatus` | 灯塔运行状态 | 展示可用、降级、离线 |
| `motionPreference` | 系统偏好 / 用户设置 | 控制动态强度 |
| `sensoryPreference` | 用户设置 | 控制声音与强氛围 |

## 3. 输出层

| 层 | 作用 | 降级 |
| --- | --- | --- |
| 光线层 | 场景色温、明暗、状态提示 | 静态背景色 |
| 粒子层 | 世界活性、迁移方向、空间深度 | 关闭 |
| 材质层 | 档案、地图、时间河等空间质感 | 低纹理纯色 |
| 声音状态 | 环境音、提示音、音乐可用性 | 默认静音 |
| AI 信标 | 灯塔可用、思考、离线、回退 | 文本状态 |

## 4. 场景氛围

| 场景 | 氛围关键词 | 声音建议 |
| --- | --- | --- |
| Home | 抵达、开阔、召唤 | 轻提示音，默认关闭 |
| Atlas | 星图、关系、远近 | 低频环境脉冲，opt-in |
| Node | 安静、聚焦、房间感 | 无默认背景音乐 |
| Paths | 行进、节奏、完成 | 步进提示音，opt-in |
| Timeline | 流动、回声、年月 | 极轻时间流环境音 |
| Archive | 纸页、档案、沉淀 | 翻页 / 抽屉音效，opt-in |
| Lighthouse | 扫描、观测、信标 | 信标音，严格可关闭 |

## 5. 边界

- 不自动播放声音。
- 不用强动效妨碍阅读。
- 不用大面积深色遮罩掩盖内容不足。
- 不把氛围状态写死在组件内。
- 所有氛围由统一 registry 或 runtime state 驱动。

## 6. 验收

- 无氛围层时页面仍完整可读。
- reduced-motion 关闭复杂动效。
- reduced-sensory 关闭声音和强闪烁。
- mobile 下不牺牲首屏速度。
