# WorldOS Archive 档案馆规格

## 1. 目标

Archive 必须像档案馆 / 记忆库，支持检索、沉淀和回看，而不是标签归档页。

## 2. 必备结构

- 分区：主题、年代、状态、可见性。
- 检索：中文优先，可低门槛使用。
- 密度视图：允许快速扫视。
- 节点出口：进入 Node、Timeline、Paths。
- 权限说明：私密内容只展示边界，不泄露原文。

## 3. 动态要求

- 分区抽屉和档案感可轻量表达。
- 不用复杂动效妨碍检索。
- reduced-motion 下保持普通检索与列表。

## 4. 验收

- 能快速找内容。
- 能看出“记忆库”而非标签页。
- 权限边界清楚。
