> 制定日期：2026-07-08
> 适用范围：古月浮屿 / WorldOS 全部数字资产
> 优先语言：中文

---

## 一、协议目的

这份文档回答一个问题：**如果我不在了，这个世界怎么处理？**

古月浮屿不是博客，不是作品集，而是一个持续生长的个人数字世界。它包含代码、内容、数据、媒体和域名。这份协议确保这个世界可以被导出、被理解、被传承，而不是在某个时刻静默消失。

## 三、导出包打开方式

### 3.1 一键导出

```bash
npm run export:world
```

生成的文件：`exports/worldos-export-YYYY-MM-DD.zip`

### 3.2 导出包含内容

- 83 个 Markdown 文件（全部公开内容）
- 37 个 JSON 文件（节点、路径、关系、事件等结构化数据）
- 6 张 SVG 封面
- 不含 `node_modules`、`.next`、`.git`、`scripts/_legacy/`

### 3.3 打开导出包

导出包不依赖任何运行时环境，解压后即可阅读：

```bash
unzip worldos-export-YYYY-MM-DD.zip -d world-archive
cd world-archive
# 用任意 Markdown 阅读器打开 content/ 下的文件
# 用任意 JSON 查看器打开 data/ 下的文件
```

如果需要以网站形式查看，可部署整个 Git 仓库：

```bash
git clone <仓库地址>
cd World
npm install
npm run build
npm run start
```

## 四、权限边界

古月浮屿有严格的权限分层：

| 层级 | 可见性 | 说明 |
|---|---|---|
| 公开层 | `public` | 任何人可访问，导出包只包含此层 |
| 私密层 | `private` / `family` / `partner` / `vault` / `sealed` / `silent` | 服务端权限守门，永不进入导出包 |

**导出包只包含 `public` 可见性的内容。** 私密内容存储在 Git 仓库中，由 middleware 服务端拦截，不会泄漏到公开层。

如果需要销毁私密数据：
1. 删除 `data/` 目录下所有标记为非 `public` 的条目
2. 删除 `content/` 目录下对应私密内容文件
3. 运行 `npm run check:content` 确认无残留
4. 提交 Git 记录销毁操作

## 五、内容许可

### 5.1 公开内容

古月浮屿的公开内容（`content/` 目录下的 Markdown 文件）采用 **CC BY-NC 4.0** 许可：
- 可自由分享和改编
- 必须署名
- 不得用于商业目的

### 5.2 代码

源代码采用 **MIT** 许可，详见仓库根目录 `LICENSE` 文件（如有）。

### 5.3 媒体

封面 SVG 由项目自制，采用与公开内容相同的 CC BY-NC 4.0 许可。

## 六、域名续费

当前状态：**域名未注册**。

当域名注册后：
1. 记录域名注册商、到期日期、DNS 配置
2. 设置自动续费
3. 在此文档中更新域名信息
4. 将域名续费提醒加入日历（提前 30 天）

如果域名过期：
1. 部署平台会自动显示 404 或部署 URL 回退
2. 导出包仍然可用（不依赖域名）
3. 在域名注册商处尝试赎回

## 七、应急处理

### 7.1 如果我无法维护

1. 导出包在 `exports/` 目录中，随时可用
2. Git 仓库包含完整历史，可克隆保存
3. 部署平台（如已上线）会在免费额度内继续运行，直到额度用尽
4. 域名（如已注册）在到期前仍然有效

### 7.2 如果需要下线

1. 在部署平台删除项目
2. 取消域名续费
3. 导出最终版本的 Git 仓库和导出包
4. 在此文档中记录下线日期和原因

### 7.3 如果需要传承给他人

1. 将 Git 仓库所有权转移
2. 传递域名管理权
3. 传递部署平台账号
4. 交接此传承协议文档
5. 新维护者需阅读 `README.md` 和 `CONTRIBUTING.md`

## 八、世界月报与修复日

古月浮屿有两个维护仪式：

| 仪式 | 频率 | 说明 |
|---|---|---|
| 世界月报 | 每月一次 | 记录当月新增节点、路径、关系和事件 |
| 修复日 | 每周一次 | 低心力维护窗口，修复小问题、补充内容 |

这些仪式记录在 `data/core/world-events.json` 中，确保世界有连续的生命轨迹。
